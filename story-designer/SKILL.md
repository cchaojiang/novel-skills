# Story Designer Skill

全本主线故事设计，用于创建小说的核心故事线和剧情架构。

**目标：支持800章×4000字的高质量玄幻修仙小说**

---

## 适用场景

- 设计小说的核心主线故事
- 设计各卷的故事走向
- 设计主要冲突和转折点
- 与 world-builder（世界观）配合使用
- **必须支持800章×4000字的丰富内容**

---

## ⚠️ 核心质量要求

### 1. 800章规模要求

```
总字数：800章 × 4000字 = 3,200,000字（320万字）

分卷规划：
- 方案1：16卷 × 50章 = 800章
- 方案2：20卷 × 40章 = 800章
```

### 2. 故事架构原则

**❌ 不要关注：**
- 章节细节描写
- 对话内容
- 场景渲染
- 水字数内容

**✅ 只需要输出：**
- 故事架构（Story Architecture）
- 核心冲突设计
- 剧情线规划
- 人物命运走向
- 势力演变
- 伏笔布局

**干货要求：**
- 每卷都是实质性剧情推进
- 无重复套路
- 每条剧情线都有明确目标和结果
- 伏笔必须有回收闭环

### 2. 每卷必须包含

```markdown
# 第X卷：卷名

## 基本信息
- 卷名：
- 副标题：
- 章节范围：
- 主题：
- 地点：
- 目标境界：

## 核心剧情（详细描述，800字以上）
本卷的核心故事情节，需要详细展开，支撑15-20章的内容

## 主要人物
- 主角：本卷行动
- 主要配角：本卷出场/成长
- 反派：本卷阴谋/对决

## 本卷爽点（3-5个）
1. xxx（详细描述）
2. xxx

## 本卷伏笔（2-3个）
1. xxx（埋入点）
2. xxx

## 本卷剧情线（5-8条支线）
1. 主线：xxx
2. 支线1：xxx
3. 支线2：xxx
...

## 本卷结局
本卷结尾的剧情走向

## 承上启下
- 承接上一卷：
- 引出下一卷：
```

### 3. 故事设计原则

**❌ 禁止出现的问题：**
- ❌ 前面卷内容详细，后面卷内容缩水
- ❌ 前期爽点密集，后期爽点稀少
- ❌ 前期伏笔丰富，后期伏笔断更
- ❌ 前期人物出彩，后期人物脸谱化

**✅ 质量均匀要求：**
- ✅ 每卷核心剧情字数均衡（800-1000字/卷）
- ✅ 每卷爽点数量均衡（3-5个/卷）
- ✅ 每卷伏笔数量均衡（2-3个/卷）
- ✅ 每卷剧情线数量均衡（5-8条/卷）
- ✅ 每卷大战场景（2-3场/卷）
- ✅ 配角在每卷都有表现机会
- ✅ 后期卷同样有高潮和转折

**丰富度要求：**
- ✅ 每卷至少5条剧情线
- ✅ 每卷至少3个主要配角成长
- ✅ 每卷至少2个势力兴衰
- ✅ 每卷至少1个秘境/遗迹探索
- ✅ 每卷至少3场大战

**质量要求：**
- ✅ 800章剧情不重复、不水字数
- ✅ 主角每50章有一次重大突破
- ✅ 每10章有一个小高潮
- ✅ 每50章有一个大高潮
- ✅ 伏笔贯穿全文，形成闭环

## 构建内容

### 1. 主线剧情设计（支撑800章）

```
核心冲突：
- 主线矛盾1条（贯穿800章）
- 支线矛盾5-8条（每100章一个完整支线）

主角成长弧线：
- 起源：凡人之躯（1-50章）
- 觉醒：修炼入门（51-150章）
- 崛起：势力建立（151-300章）
- 抗争：对抗强敌（301-500章）
- 巅峰：飞升仙界（501-650章）
- 归真：成就道祖（651-800章）

每阶段必须包含：
- 重大转折（2-3个）
- 势力变化（2-3个）
- 人物命运（5-8个）
- 秘境/遗迹（2-3个）
```

### 2. 卷剧情概要（16-20卷）

```
每卷需要包含：
- 卷名 + 副标题
- 章节范围（40-50章/卷）
- 本卷核心矛盾
- 本卷主要人物（主角+5-8个配角）
- 本卷目标
- 本卷爽点（3-5个详细描述）
- 本卷伏笔（2-3个）
- 本卷剧情线（5-8条）
- 本卷结局
- 承上启下
```

### 3. 关键转折点（800章跨度）

```
每50章一个重大事件：
- 第50章：第一个重大转折
- 第100章：势力格局变化
- 第150章：主角危机
- 第200章：结盟/背叛
- ...
- 第800章：大结局

每10章一个小高潮：
- 战斗/突破/发现/反转
```

### 4. 伏笔体系（贯穿800章）

```
伏笔设计原则：
- 前期埋入（1-200章）：至少20个伏笔
- 中期呼应（201-500章）：呼应前期伏笔15个
- 后期收束（501-800章）：收束所有伏笔

伏笔类型：
- 人物命运伏笔
- 势力兴衰伏笔
- 功法来源伏笔
- 身世秘密伏笔
- 终极BOSS伏笔
```

### 5. 势力格局演变

```
势力变化：
- 前期：主角起源势力 → 中期：建立自己的势力 → 后期：称霸/飞升

每个阶段需要：
- 2-3个势力兴起
- 2-3个势力衰落
- 1-2个势力覆灭
- 隐藏势力浮现
```

## 工作流（8步）

```
Step 1: load_inputs           加载用户输入 + 世界观设定
Step 2: design_main_arc     StoryArchitect 设计800章主线架构
Step 3: design_volume_structure StoryArchitect 规划16-20卷
Step 4: design_all_volumes StoryArchitect 设计每卷详情（800字+/卷）
Step 5: design_turning_points 设计关键转折点（16个）
Step 6: design_foreshadow  设计伏笔体系（20+个伏笔）
Step 7: StoryReviewer       校验故事逻辑/丰富度/伏笔
Step 8: save_story          保存故事设定
```

```markdown
## 主线剧情
主角林逸原本是废物，因绝焰之体觉醒踏上修炼之路。过程中发现父亲死亡真相，最终反抗仙界之主。

## 各卷概要
- 第1卷：觉醒篇 - 绝焰之体觉醒
- 第2卷：成长篇 - 势力建立
- 第3-6卷：抗争篇 - 正邪大战
- 第7-12卷：逆天篇 - 反抗仙界

## 关键转折
- 第6卷：发现父亲死亡真相
- 第9卷：突破半步真仙
- 第12卷：成就道祖

## 伏笔体系
- V001: 绝焰之体真正来源（埋入：第1卷，呼应：第7卷）
- V002: 仙界之主阴谋（埋入：第3卷，呼应：第8卷）
```
