# Novel Skills

小说生成相关OpenClaw Skills，包含6个技能：

| Skill | 职责 |
|-------|------|
| **story-designer** | 故事主线设计（10卷剧情架构） |
| **world-builder** | 世界观设定（境界体系、修炼体系、势力格局） |
| **novel-planner** | 卷大纲生成（每卷500字以上剧情） |
| **chapter-planner** | 章节大纲生成（每章10项信息） |
| **chapter-optimizer** | 章节大纲优化（情节丰富、爽点增强） |
| **chapter-writer** | 章节正文生成（3000-4000字/章） |

---

## 工作流

```
story-designer → world-builder → novel-planner → chapter-planner → chapter-optimizer → chapter-writer
```

---

## 使用方法

这些Skills需要配合OpenClaw使用，详细说明请参考各Skill目录下的SKILL.md文件。
